# server_python
Server Python

# Prerequisites

1. Python in version 3.6. minimum
2. The "Flask" framework installed
Command line for installation : 
```cmd 
$ pip install flask 
```
or 
```
$ python -m pip install flask 
```
or 
```
$ python3 -m pip install flask 
```
